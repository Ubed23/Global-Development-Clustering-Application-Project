# 🌍 Global Development Clustering Application

An interactive Machine Learning application that groups countries based on their socio-economic and health indicators using **K-Means Clustering**. The project helps users identify development patterns through data visualization and predictive analytics.

---

## 🚀 Features

- 📊 Interactive data visualization
- 🌍 Country clustering using K-Means
- 📈 Cluster distribution analysis
- 🔍 Explore development patterns
- ⚡ Fast and user-friendly interface
- 📉 Data-driven insights for decision making

---

## 🛠️ Tech Stack

- NumPy
- Scikit-learn
